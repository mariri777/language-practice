# Language Practice

中国語・韓国語の学習ツール集。GitHub Pagesで公開。

🌐 **[mariri777.github.io/language-practice](https://mariri777.github.io/language-practice/)**

## ツール

| ツール | 内容 |
|---|---|
| [汉语日记](chinese-diary.html) | HSK 1〜4対応の1日1パッセージ書写・音読・タイピングトレーニング |
| [中国語アウトプット練習](chinese-practice.html) | HSK級別の日本語→中国語作文ドリル |
| [韓国語アウトプット練習](korean-practice.html) | 日本語→韓国語作文ドリル |

## 特徴

- すべて単一HTMLファイル（外部依存なし）
- 端末内蔵のTTSで発音再生
- 進捗はlocalStorageに保存
- ダーク／ライトモード切替
- レスポンシブ対応

## ローカルでの起動

```bash
# ファイルを直接ブラウザで開くか、簡易サーバーで起動
python3 -m http.server 8000
# → http://localhost:8000
```
